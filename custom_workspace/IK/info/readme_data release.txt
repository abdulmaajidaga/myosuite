EMG Data:

The EMG files contain EMG data (in Volt (V)). The table is divided into 12 columns for each of the 12 measured muscles
column1: M. deltoideus pars clavicularis (DC)
column2: M. Biceps brachii (BB)
column3: M. Triceps brachii (TB)
column4: M. Flexor digitorum superficials (FDS)
column5: M. Extensor digitorum (ED)
column6: M. Brachioradialis (BR)
column7: M. Flexor carpi ulnaris (FCU)
column8: M. Extensor carpi ulnaris (ED)
column9: M. Pronator teres (PT)
column10: M. Flexor carpi radialis (FCR)
column11: M. Abductor pollicis brevis (APB)
column12: M. Abductor digiti minimi (ADM)

 
 The rows correspond to the data points for each frame (capture rate: 2000Hz)
 
 
 
 Kinematic data:
 
 The kinematic data files contain the position data of thorax, upper- and forearm markers. The table is divided into 63 columns with 3 columns (x, y, z position) for each of the 21 markers:
 marker1: WRA, wrist - thumb side
 marker2: WRB, wrist - little finger side
 marker3: RU_1, marker cluster forearm, marker no. 1
 marker4: RU_2, marker cluster forearm, marker no. 2
 marker5: RU_3, marker cluster forearm, marker no. 3
 marker6: RU_4, marker cluster forearm, marker no. 4
 marker7: ELB_M, elbow medial
 marker8: ELB_L, elbow lateral
 marker9: H_1, marker cluster upper arm, marker no. 1
 marker10: H_2, marker cluster upper arm, marker no. 2
 marker11: H_3, marker cluster upper arm, marker no. 3
 marker12: H_4, marker cluster upper arm, marker no. 4
 marker13: H_5, marker cluster upper arm, marker no. 5
 marker14: H_6, marker cluster upper arm, marker no. 6
 marker15: SA_1, behind acromial joint
 marker16: SA_2, in front of acromial joint
 marker17: SA_3, acromial joint
 marker18: CS_1, marker cluster thorax, marker no. 1
 marker19: CS_2, marker cluster thorax, marker no. 2
 marker20: CS_3, marker cluster thorax, marker no. 3
 marker21: CS_4, marker cluster thorax, marker no. 4
 
 The rows (from 3rd row) correspond to the data points (marker positions (in mm)) for each frame (capture rate: 200 Hz)
 1st row: marker names
 2nd row: unit (mm)
 
 
 
 
 Naming the trials:
 Each trial name consists of 3 numbers (e.g. 10_8_3)
 first number (in this case 10): subject or patient number. Stroke patients are indicated by an S (e.g. S10)
 second number (in this case 8): task number
 third number (in this case 3): trial number
